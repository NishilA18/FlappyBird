# Flappy Bird

A browser-playable Flappy Bird-style arcade game built with plain HTML, CSS, and JavaScript.

## Play

Open `index.html` in a browser, then click, tap, or press Space to flap.

## Features

- Responsive canvas gameplay
- Procedural pipes and scrolling ground
- Score and best score tracking
- Keyboard, mouse, and touch controls
- Lightweight generated visuals and Web Audio sound effects

## Files

- `index.html` - page markup
- `style.css` - layout and interface styling
- `script.js` - game loop, collisions, drawing, scoring, and audio
